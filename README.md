# mcu_Scaffolding
1：可以切换多种时钟源，并可以实现不同部分上下电 2：不同低功耗模式，定时器唤醒和外部io唤醒，不同的电源域配置 3：电源动态配置，根据外设需求动态配置电源 4：dma池动态配置，根据需求申请，可以支持抢占 5：中断动态配置，根据需求申请，可以支持抢占 6：shell 7：定长和不定长的内存池 8：freertos信息统计 9：栈回溯信息 10：文件系统，可以保存栈回溯信息 11：内存和cpu信息dump，可以保存到flash和通过串口导出；外设寄存器只读取，难以正确写入（只读、需要特定顺序、有特殊影响） 12：内存和cpu信息dump文件恢复到cpu中，通过gdb attach到芯片，可以查看所有线程栈帧和变量
